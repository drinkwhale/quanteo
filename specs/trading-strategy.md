# CCI + 박병창 매매기법 종합 전략 문서

> 이 문서는 자동매매 시스템 구현을 위한 전략 스펙입니다.
> Claude Code가 이 문서를 읽고 백테스트 / 실매매 코드를 작성하는 데 사용합니다.

## 0. 문서 목적 및 범위

- 대상 종목: SK하이닉스(000660) 등 한국 주식 (다른 종목으로도 확장 가능)
- 데이터 소스: Toss Open API
- 타임프레임: 월봉, 주봉, 일봉, 60분봉 (4단 계층 분석)
- 보조지표: CCI(20, 20) — period=20, signal=20
- 매매 철학: 박병창 「매매의 기술」 매수 3원칙 / 매도 2원칙 + 장중 시그널 4유형
- 목표: 위 원칙을 정량적 조건으로 변환하여 백테스트 가능한 전략으로 구현

## 1. CCI 지표 정의

### 1.1 계산 공식

```
TP (Typical Price) = (고가 + 저가 + 종가) / 3
SMA = TP의 20일 단순이동평균
MD (Mean Deviation) = |TP - SMA| 의 20일 평균
CCI = (TP - SMA) / (0.015 * MD)

Signal = CCI의 20기간 이동평균
```

- `period = 20`, `signal_period = 20`
- 변수명 규칙: `cci`(CCI 본선), `cci_signal`(시그널선)

### 1.2 해석 기준

| CCI 값    | 의미               |
| --------- | ------------------ |
| +100 이상 | 과매수             |
| 0         | 중립 / 추세 전환점 |
| -100 이하 | 과매도             |
| +200 이상 | 강한 과열          |
| -200 이하 | 강한 공포          |

### 1.3 크로스 신호

- **골든크로스**: `cci`가 `cci_signal`을 상향 돌파
  - -100 이하에서 발생 → 신뢰도 최고
  - 0 부근 → 신뢰도 보통
  - +100 이상 → 신뢰도 낮음 (이미 과매수)
- **데드크로스**: `cci`가 `cci_signal`을 하향 돌파
  - +100 이상에서 발생 → 신뢰도 최고
  - 0 부근 → 신뢰도 보통
  - -100 이하 → 신뢰도 낮음 (이미 과매도)

### 1.4 멀티 타임프레임 계층 구조

```
월봉 CCI  → 대세 방향 결정 (1~2년 추세)
주봉 CCI  → 중기 추세 확인 (3~6개월)
일봉 CCI  → 매매 기준선 (핵심, 가장 중요)
60분봉 CCI → 진입 타이밍 포착 (보조)
```

원칙: **상위 타임프레임이 방향을 결정하고, 하위 타임프레임이 타이밍을 결정한다.**
하위 타임프레임 신호는 상위 타임프레임 방향과 일치할 때만 유효하다.

## 2. 박병창 매매기법 핵심 철학

1. 이동평균선은 5일선, 20일선만 사용한다 (60일/120일은 분석에서 배제).
2. 거래량이 모든 판단의 절대 기준이다.
3. 매도 타이밍이 매수 타이밍보다 중요하다 (실패 시 수익이 손실로 전환됨).
4. 분할 매수 + 분할 매도를 원칙으로 한다.
5. 원칙에 맞지 않는 시장 상황에서는 매매하지 않고 쉰다.
6. 본질은 "매수세(황소)가 강한가, 매도세(곰)가 강한가"의 판단이다.

## 3. 매수 3원칙

### 매수 제1원칙 — 5일선 위 (급등주 매수)

조건:

- 주가가 5일선 위에서 상승 중
- 비교 대상 음봉이 없어 50% 룰 적용 불가 → 장중 타이밍으로 판단

타이밍:

- **오전**: 10시 이전, 시가 아래로 일시 하락 후 거래량 증가하며 시가 재돌파
- **오후**: 14시 이후, 장중 조정 마무리 후 거래량 증가하며 재상승

제외 조건:

- 오전 강세 → 10시 이후 하락 전환 시 매수 대상에서 제외
- 오전에 매수 신호가 없었던 종목은 장중 거래하지 않고 오후 14시 이후 재관찰

### 매수 제2원칙 — 5일선과 20일선 사이 (눌림목 매수)

조건 (모두 충족):

1. 하락 직전 강하게 급등한 종목일 것
2. 하락 시 거래량이 급격히 감소할 것 (고점 거래량 대비 40% 이하 권장)
3. 하락폭이 직전 상승폭의 50% 이내일 것
4. 재상승 당일 오전부터 거래량이 크게 증가하며 양봉 발생
5. 강한 종목일수록 10일선 전후에서 조정을 마무리

시간대:

- 강세 종목: 오전 10시 이전 매수
- 약세 종목: 오후 14시 30분 이후, 거래량 증가 장대양봉 확인 후 매수

금지: 거래량 증가하며 음봉 발생 시 매수 절대 금지.

### 매수 제3원칙 — 20일선 아래 (급락 후 저점 매수)

조건:

1. 거래량이 최대한 감소한 종목을 선별
2. 거래량 급증과 함께 양봉 또는 십자형 발생
3. 반등 후 재하락 시 직전 저점을 하향 이탈하지 않으면 추가 매수

적용 대상:

- 이벤트성(노이즈) 하락 시장: 시장 대표주, 테마 주도주 우선
- 외부 이슈로 동반 하락 시 거래량이 급감한 종목

금지:

- 거래량 증가하며 하락하는 종목은 매수 금지
- "많이 떨어졌다"는 이유만으로 매수 금지

## 4. 매도 2원칙

### 매도 제1원칙 — 5일선 위에 있을 때

신호 및 비율:

- 거래량 급증 + 음봉 발생 → 30~50% 매도 (1차 분할 매도)
- 거래량 폭증 + 십자형 발생 → 30% 매도
- 5일선 이탈 + 20일선 하향 시 → 나머지 전량 매도

매도 비율 가이드:

- 상승세가 강할수록 → 매도 비율을 작게
- 상승세가 약할수록 → 매도 비율을 크게

### 매도 제2원칙 — 5일선과 20일선 사이에 있을 때

신호:

- 5~20일선 사이에서 거래량 증가 + 음봉 발생 → 2차 매도

특별 규칙 (헤드앤숄더 연계):

- 20일선 바로 위에서 거래량 증가 + 장대음봉 발생 시
  → 추세 전환 가능성이 크므로 전량 매도

거래량 없이 하락하는 경우:

- 주도 세력이 매도하지 않는다는 의미이므로 보유 유지 가능 (조정으로 판단)

## 5. 장중 시그널 4가지 유형

판단 기준 시간: 오전 10시, 오후 14시(또는 14:30) 두 지점이 핵심 분기점.

### ①번 유형 — 강세 매수 신호

패턴: 전일 종가 근방 출발 → 10시 이전 잠시 하락 → 곧바로 상승 전환 → 장중 박스권 유지 → 장 마감 거래량 증가 장대양봉

행동:

- 오전 하락 시 매수 또는 오후 14시 이후 재상승 시 매수
- 거래량이 클수록 신뢰도 상승
- 주의: 다음 날 매수 시 당일 저가보다 높은 가격에 매수하게 됨 (당일 장중 포착이 핵심)

### ②번 유형 — 매도/주의 신호

패턴: 오전 하락 후 상승 (①번과 동일) → 장 후반 매물 급격히 출회 → 시가 아래로 하락 → 장 마감까지 하락 지속 → 위꼬리 달린 음봉

행동:

- 5일선 위에서 발생 → 1차 매도(30~50%)
- 5~20일선 사이에서 발생 → 추세 전환 가능성 크므로 적극 매도 검토
- 20일선 붕괴 동반 시 → 전량 매도
- 매수 타이밍을 다음 날로 유보 (당일은 매수 금지)

### ③번 유형 — 오후 반등 매수 신호

패턴: 오전부터 하락 → 장중 음봉 상태 지속 → 오후 14시~14:30 이후 거래량 급증하며 반등 → 시가 위로 마감 (아래꼬리 양봉/십자형)

행동:

- 오후 14:30 이후 거래량 증가 확인 후 매수
- 매수 제2원칙 또는 제3원칙 근거로 사용
- 거래량이 최근 평균 대비 클수록 확신을 갖고 매수
- 약세 종목은 "전강후약"이 많으므로 오전이 아닌 오후에 확인 후 매매 판단

### ④번 유형 — 강한 매도 신호 (최우선 경계)

패턴 A (강세주 고점): 오전 급등 후 즉시 하락 전환 → 장중 내내 하락 → 위꼬리 장대음봉
패턴 B (약세주 지속 하락): 오전부터 하락 → 장 마감으로 갈수록 매도세 강화 → 장대음봉

행동:

- 5일선 위 발생 → 1차 매도
- 20일선 근처 발생 → 전량 매도 (20일선 붕괴 후 급락 가능성 높음)
- 매수 절대 금지
- 다음 날도 하락 확률이 매우 높음

### 특수 패턴 — 45도 각도 점진적 하락 (최악의 패턴)

정의: 장중 내내 완만한 각도로 지속 하락, 중간중간 거래량도 일부 실리는 패턴.

판단 근거: 매도자가 오랜 시간에 걸쳐 꾸준히 매도 중이라는 의미. 반등 시도 시마다 매물이 출회되어 상승 탄력이 약화됨.

행동:

- 신규 매수 절대 금지
- 보유 중이면 즉시 매도 검토
- 이 패턴에 해당하는 종목은 거래 종목에서 제외

## 6. 거래량 판단 기준 (정량 임계값)

```
매수 허용 거래량:
- volume > prev_down_candle_volume * 0.5  (전일 음봉 거래량의 50% 초과)
- volume > volume_ma20 * 1.5  (조정 시 대비 증가)
- volume > volume_ma20 * 2.0  (강한 신호, 급락 후 반등 시)

매수 금지 거래량:
- 거래량 증가하며 하락하는 캔들
- 거래량 없이 상승하는 캔들 (volume < volume_ma20 * 1.0)
- 조정 구간 거래량이 직전 고점 거래량의 40% 이상 (급감하지 않음)

매도 신호 거래량:
- volume > volume_ma20 * 2.0 + 음봉/십자형
- 20일선 위에서 volume > volume_ma20 * 2.0 + 장대음봉 → 전량 매도
```

## 7. 헤드앤숄더 패턴 (추세 전환 보조 신호)

- 하락 전환: 왼쪽 어깨 → 머리(고점) → 오른쪽 어깨. 오른쪽 어깨 형성 시 거래량 감소 + 직전 하락폭 50% 초과 양봉 실패 → 추세 하락 전환, 전량 매도.
- 만약 오른쪽 어깨 형성 중 거래량 증가 + 직전 하락폭 50% 초과 양봉이 나오면, 그것은 어깨가 아니라 눌림목이며 추세 상승이 지속된다.
- 상승 전환(역헤드앤숄더): 거래량 증가 + 장대양봉으로 왼쪽 고점을 돌파하면 추세 상승 전환.
- 적용 타임프레임: 주봉 권장 (중기 추세 판단용).

## 8. CCI + 박병창 통합 매매 로직

### 8.1 진입(매수) 의사결정 트리

```
1단계 (월봉 CCI):
  IF 월봉_CCI > 월봉_CCI_signal AND 월봉_CCI > 0:
      → 대세 상승, 매매 허용 영역. 2단계로.
  ELSE:
      → 매매 금지, 관망.

2단계 (주봉 CCI):
  IF 주봉_CCI > 주봉_CCI_signal:
      → 중기 추세 양호. 3단계로.
  ELSE:
      → 비중 축소 또는 관망.

3단계 (일봉 CCI + 박병창 매수원칙):
  IF 일봉_CCI > 일봉_CCI_signal:
      IF price > ma5:
          → 매수 제1원칙 후보 (장중 시그널 ①/③번 확인)
      ELIF ma5 < price < ma20:
          → 매수 제2원칙 후보 (눌림목, 거래량 조건 확인)
  ELIF price < ma20 AND 일봉_CCI < -100:
      → 매수 제3원칙 후보 (급락 저점, 거래량 급감→급증 확인)
  ELSE:
      → 매수 보류.

4단계 (60분봉 CCI + 장중 시그널):
  IF 60분봉_CCI > 60분봉_CCI_signal AND 60분봉_CCI < 100:
      IF 장중유형 in ["①번", "③번"] AND volume > volume_ma20 * 1.5:
          → 매수 실행 (분할 매수 원칙 적용)
```

### 8.2 청산(매도) 의사결정 트리

```
즉시 전량 매도 조건 (OR 조건, 하나라도 충족 시):
  - 45도 각도 점진적 하락 패턴 감지
  - price < ma5 AND price < ma20
  - 20일선 근처에서 ④번 유형 발생
  - 20일선 바로 위 거래량 급증 장대음봉 (매도 제2원칙 특별 규칙)
  - 일봉_CCI 가 +100 이상에서 데드크로스 + ②번 유형 동시 발생

분할 매도(30~50%) 조건:
  - price > ma5 AND volume > volume_ma20 * 2.0 AND candle == "음봉"
  - price > ma5 AND volume > volume_ma20 * 2.0 AND candle == "십자형" → 30%
  - ma5 < price < ma20 AND volume > volume_ma20 * 1.5 AND candle == "음봉" → 50%

보유 유지 조건:
  - 거래량 없이 하락 (주도세력 매도 아님)
  - 일봉_CCI 골든크로스 유지 + 월봉/주봉 CCI 골든크로스 유지
```

### 8.3 신뢰도 스코어링 (선택적 가중치 시스템)

각 조건 충족 시 +1점, 매도 신호 조건은 -1점으로 환산하여 합산 점수로 포지션 크기를 결정하는 것을 권장.
양성 항목 합계 최대 8점 (1+1+2+1+1+1+1).

```
+1: 월봉 CCI 골든크로스
+1: 주봉 CCI 골든크로스
+2: 일봉 CCI 골든크로스 (가중치 2배 — 매매 기준선)
+1: 60분봉 CCI 골든크로스
+1: 거래량 volume_ma20 대비 1.5배 이상
+1: 장중 유형 ① 또는 ③번
+1: 이동평균선 정배열 (5일선 > 20일선)
-2: 일봉 CCI 데드크로스
-1: 45도 각도 하락 패턴
-2: ④번 유형 발생

점수 해석:
  7점 이상 → 적극 매수 (분할 매수 1차)
  4~6점   → 소극 매수 (소액 분할)
  0~3점   → 관망
  음수    → 매도 검토
```

## 10. 백테스트 설계 가이드 (구현 시 참고)

1. 4개 타임프레임(월/주/일/60분) 데이터를 동기화하여 일봉 기준으로 리샘플링.
2. 각 일봉 시점마다 상위 타임프레임(월/주) CCI 상태를 lookup.
3. 매수/매도 신호는 "신호 발생 다음 봉 시가"에 체결하는 것을 기본으로 함 (미래참조 방지).
4. 수수료 및 슬리피지 반영 (한국 주식 기준 매수 0.015% + 매도 0.015% + 거래세 0.18% 권장).
5. 분할 매수/매도는 포지션을 0~100% 사이 비율로 관리 (전량 단위가 아닌 비율 단위로 시뮬레이션).
6. 성과 지표: 승률, 평균 손익비, MDD(최대낙폭), 샤프지수, 거래 횟수.
7. 파라미터 최적화 시 과최적화(overfitting) 주의 — walk-forward 검증 권장.

현재가: 2,939,000원

## 12. 용어 사전 (Glossary)

| 용어              | 정의                                                |
| ----------------- | --------------------------------------------------- |
| 눈림목            | 상승 추세 중 일시적 조정 (5~20일선 사이)            |
| 황소              | 매수 세력                                           |
| 곰                | 매도 세력                                           |
| 50% 룰            | 직전 캔들 변동폭의 50%를 기준으로 한 매수/매도 판단 |
| 장대양봉/장대음봉 | 몸통이 크고 꼬리가 짧은 강한 추세 캔들              |
| 도지형(십자형)    | 시가와 종가가 거의 같은, 매수/매도 균형 캔들        |
| 정배열            | 5일선 > 20일선 (단기 이평이 위에 위치)              |
| 역배열            | 5일선 < 20일선                                      |

## 13. 구현 시 주의사항 (Claude Code 참고)

- 이 문서의 모든 임계값(예: `volume_ma20 * 1.5`)은 박병창 원문의 정성적 표현("거래량이 크게 증가")을 정량화한 것으로, 백테스트 결과에 따라 파라미터 튜닝이 필요합니다.
- 장중 시그널 4유형 판단은 "당일 마감 전까지는 확정되지 않는다"는 원문의 핵심 경고를 반드시 코드에 반영해야 합니다 (look-ahead bias 방지).
- 토스증권 등 일부 플랫폼은 CCI 본선/시그널선 색상이 반대로 표시되므로(보라=CCI, 파란=시그널), 데이터 파싱 시 색상이 아닌 라벨/값 기준으로 구분해야 합니다.
- 코드 작성 전 단계로, 이 문서는 전략 정의(WHAT)만 다루며 구현(HOW)은 별도 단계에서 진행합니다.
