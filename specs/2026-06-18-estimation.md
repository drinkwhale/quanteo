## 리스크 요인

| 리스크                          | 확률 | 영향 | 설명                                                      |
| ------------------------------- | ---- | ---- | --------------------------------------------------------- |
| KIS WS 불안정 / TR_ID 오류      | 높음 | 크다 | 환경별 TR_ID가 다르고 공식 문서가 불완전한 경우 있음      |
| vps 장 운영 시간 제약           | 높음 | 중간 | 통합 테스트는 주중 주간만 가능                            |
| TS 대시보드 스택 미확정         | 중간 | 중간 | Next.js vs Vite vs plain React 결정 전까지 P6 추정 불확실 |
| asyncio + FastAPI WS 동시 운영  | 낮음 | 중간 | uvicorn 이벤트 루프 공유 방식 선택 필요                   |
| Telegram rate limit 구현 복잡도 | 낮음 | 낮다 | 설계서에 priority queue + 1s 간격 명시됨                  |
